How to code your data

You can code documents on the Workdesk screen. First, select the document you want to code from the drop-down list and it will load on the screen. The codes will be available on the right side of the screen.

To code a segment of the text select the segment and click on the code you want to attach to the segment.

Segments may overlap. Code colours will be mixed.

How to delete coding of a segment

Text marked with one code

Click on the coded segment and hit on the Remove code button.

Text marked with several codes

Click on the coded segment. A dialog box appears. Select the code to be removed. Confirm by hitting the Remove Selected button.
