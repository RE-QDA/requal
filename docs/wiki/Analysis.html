After you are done with coding

Once your data are coded, you can find coded parts in the Analysis screen. * On the middle part of the screen you can see all segments coded. * On the right side, you can find three filters. You can filter text segments by code, category of codes, or by document.

Working with filters

As default, all codes, categories and documents are selected, therefore all coded segments are displayed. You can select only one item from the filter list by clicking on it. In case you want to select multiple items hold the Ctrl (control) key and click on each item or click and drag over the items you want to select. reQual offers three filters:

  • Filter by code
  • Filter by category
  • Filter by document

If you want to turn off filters for codes or categories, select the empty item from the top of the list. Remember to keep selection of at least one document and one code or category to see text segments for analysis.

Downloading your analysis

You can download selected segments by clicking the CSV or HTML button at the bottom of the page with the coded segments. Segments will be downloaded in a file together with document and code information.

If you download an HTML file, it will be named report.html and you can open it in any web browser. This is an example of it:

If you download a CSV file, the name of the file will indicate the date and time of the export, for example requal_export-2022-11-26-160754.csv. There are several programs to work with when you want to explore or edit CSV files, e.g. MSExcel, Google Sheets, LibreOffice or Numbers.

Analysis table - description

As you can see the exported table consists of five columns: * doc_name - in this column you can look up the name of the analysed document * doc_id - a database identification of the document * code_name - this column shows the names of the codes you used during the analysis * code_id - a database identification of the code * segment_text - the most important column for your analysis, because you can see the segments you selected for chosen codes or categories.
